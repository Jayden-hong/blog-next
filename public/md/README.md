## Why Markdown?

- **70% smaller** than HTML (typically ~10KB vs ~35KB)
- **Structured metadata** in YAML frontmatter
- **Clean content** without styling noise
- **Perfect for LLMs** - fits better in context windows

```bash
curl https://blog.zucchini.win/md/daily-thoughts-2026-02-28.md \
  -H "Accept: text/markdown"
```
